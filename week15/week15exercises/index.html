JavaScript Exercises

  1. Create a copy of your template folder rename it to angulartemplate.
  2. In your js folder, create a new folder named 'controllers'.
  3. Go to this site. Click 'DOWNLOAD ANGULARJS', select the version that has (latest) in its name, and then select 'zip', and then hit download.
  4. Open the zip file and find the file named 'angular.js'. Copy it and paste it into the js folder of your new angulartemplate.
  5. In HTML: Add 'angular.js' to your template using a new script tag. This should go in the <head>.
  6. In HTML: Update the html tag so it has an ng-app attribute - this is known as an Angular directive.
  7. Create a copy of your new angulartemplate folder and rename it to week15exercise.
  8. Drag this new folder into your 'week15' folder inside your 'codify' folder.
  9. When working through these exercises, there are bits and pieces of the instructions that are deliberately left out. You should start thinking about what you THINK you will need in order to accomplish the request.
  10. In JS: In your app.js file, create an Angular app module. Store this module in a variable. Make sure to also update your ng-app directive in HTML with the name of your Angular application - remember, that the variable and the application name are NOT the same thing (though they can be the same word(s))
  11. In JS: On a new line, using the variable that your stored your app module in, create a controller.
  12. In JS: Add a string of data to the controller's $scope with the text: "This is an expression".
  13. In HTML: Create a div and an h1 element. Give the div the ng-controller attribute and make its value equal to the name of your controller.
  14. In JS: Using an Angular expression (the {}), display the text "This is an expression" in your h1.
  15. In HTML: Create two inputs tags for the user to give us 2 numbers, and a div to display the resulting output.
  16. In HTML: Using two-way data binding, make it so when the user updates the values in the input elements the information in the div also updates. The output should be the two numbers multipled by each other.
  17. In HTML: Add an Angular filter to the expression to change how the data is displayed.
  18. In JS: Create an array of objects and store it using $scope.
  19. In HTML: Using the ng-repeat directive, display all of the items from the array in separate HTML elements.
